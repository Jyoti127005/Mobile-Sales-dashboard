# Mobile-Sales-dashboard
Developed an interactive Mobile Sales Dashboard using Power BI to analyze and visualize mobile phone sales performance across multiple dimensions. The dashboard provides real-time insights into total revenue, units sold, average selling price, and profit trends, enabling data-driven decision-making.

Analyzed a comprehensive dataset of 3,835 observations to identify key market trends and consumer behaviors.

DAX calculations were used to growth metrics, while clean and intuitive visualizations help identify top-selling products, customer demand patterns, and sales opportunities.

This project demonstrates the effective use of Power BI for business intelligence, data modeling, and storytelling through data.
